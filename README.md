## 🚀 **Quick Start**

### Installation

```bash
# Clone and setup
git clone https://github.com/tyrannon/claude-prompter.git
cd claude-prompter && npm install && npm run build

# Set up environment
cp .env.example .env
# Add your OPENAI_API_KEY to .env

# Set up global access
./setup-alias.sh && source ~/.zshrc
```

